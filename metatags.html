Metatags

Los metatags se usan para manejar metadata, lo cual es información que engloba otra información o datos del html (se coloca en la seccion head al principio del html)

  • charset: para setear abecedarios y locaciones. Generalmente elegimos "UTF-8"
  • name y content: este atributo especifica el nombre de la información y el valor del contenido, el cual debe definirse en el siguiente atributo como content. Por ejemplo: name="author" content="VS Code" o tambien uno muy común para agregar tags que usaran los motores de busqueda como google para catalogar tu web es: name="keywords" content="HTML, meta tag, tag reference, Youtube, Brocode". La lista de valores de atributo 'name' se puede encontrar en la web. Note: If the http-equiv attribute is set, the name attribute should not be set.
  • http-equiv: el metodo refresh es util para actualizar la pagina automaticamente cada cierto tiempo. Tambien requiere valores de content
